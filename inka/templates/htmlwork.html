Hiển thị trang web với file html

Bước 1: Làm việc với file setting.py

Mở file seting.py trong folder mysite con sau đó chèn 'myapp' vào trong INSTALLED_APPS như sau:

INSTALLED_APPS = [

...

'myapp'

]

và save seting

Bước 2: Chuẩn bị file html

Trong folder myapp tạo 1 folder con có tên là templates ta sẽ lưu toàn bộ file html ở folder templates tạo 1 file index.html

Bước 3: Làm việc với views.py

Trong folder myapp có sẵn 1 file views.py bạn chỉ cần mở file ra và copy đoạn code :

from django.shortcuts import render

def home(request):

return render(request, 'index.html')

sau đó ấn save để lưu

Bước 4: Làm việc với file urls.py

Sẽ có 2 file urls.py, 1 file nằm trong folder mysite con và 1 file nằm trong folder myapp bạn có thể copy file urls.py có sẵn trong folder mysite con và paste vào folder myapp, rồi sửa.

File urls.py nằm trong folder mysite con bạn sửa như sau:

from django.contrib import admin

from django.urls import path, include

urlpatterns = [

path('admin/', admin.site.urls),

path('', include('myapp.urls'))

]

File urls.py nằm trong folder myapp bạn vừa copy từ folder mysite bạn sửa code như sau :

from django.urls import path

from . import views

urlpatterns = [

path('', views.home, name='home'),

]

sau đó lưu lại

Bước 5: Chạy thử trên server

Dùng lệnh: python manage.py runserver chạy trên server, test project bằng cách gõ 127.0.0.1:8000
