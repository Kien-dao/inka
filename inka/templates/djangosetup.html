Cài đặt django

Bước 1: Cài đặt python

Tải python tại trang python.org sau đó dùng lệnh python --version để kiểm tra version của python

Bước 2: Cài đặt môi trường ảo

Dùng lệnh python -m venv myenv để tạo môi trường ảo và dùng lệnh myenv\Scripts\activate để kích hoạt môi trường ảo

Bước 4: Cài đặt django

Dùng lệnh: pip install django để cài đặt django và dùng lệnh:django-admin --version để kiểm tra version

Tạo một dự án web với django

Bước 1: tạo 1 dự án có tên là mysite

Dùng lệnh: ddjango-admin startproject mysite để tạo 1 project mysite sau này tất cả các dữ liệu trang web đều nằm trong folder mysite này dùng lệnh:cd mysite để vào folder mysite

Bước 2: Chạy thử trên server

Dùng lệnh: python manage.py runserver chạy trên server, test project bằng cách gõ 127.0.0.1:8000 vào trình duyệt web bất kỳ ví dụ như chrome

Bước 3: Tạo 1 app có tên là myapp trong project mysite

Dùng lệnh: python manage.py startapp myapp rồi vào trong folder mysite để kiểm tra khi có folder myapp xuất hiện là đã thành công trong folder mysite gồm có 2 folder con myapp, mysite và 2 file db.sqlite3, manage.py
